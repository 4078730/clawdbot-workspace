## Gemini Robotics Trusted Tester Program

### 背景
- Googleの Gemini Robotics Trusted Tester Program に参加したい
- 申請条件：「組織がプロビジョニングしたGCPプロジェクト」が必要

### 調査結果
- 社用メール（@jp.panasonic.com）でGCPログイン可能
- しかし「組織なし」しか表示されず、組織配下でプロジェクト作成不可
- `jp.panasonic.com` は既に他組織で使用されているため、新規Cloud Identity組織作成には使用不可

### 解決策（PX クラウドCoE Q&Aより）
1. **サブドメインを取得**（例: `rfa.jp.panasonic.com`）
2. そのサブドメインで**Cloud Identity組織を作成**
3. **Workforce Identity**でPA認証連携
4. その組織配下でGCPプロジェクト作成 → Project Numberを取得

### アクション
- PX クラウドCoEに問い合わせ済み（サブドメイン取得 + GCP組織作成の手続き）
- 問い合わせ先: https://jppanasonic.sharepoint.com/sites/ONL01957
