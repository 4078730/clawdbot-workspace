## 2. Azure クラウド利用費（関数化）

### コスト構造

```
Azure費用 = Σ (環境数 × スキル数 × FT単価)
```

| 変数 | 説明 | 単位 |
|------|------|------|
| **環境数** | デプロイ先の拠点・環境 | 箇所 |
| **スキル数** | VLAスキルの種類 | 個 |
| **FT単価** | 1スキルあたりのFine-tuning費用 | 円/スキル |

### FY25実績（参考）

| 環境 | スキル数 | 備考 |
|------|----------|------|
| CXC | 5 | 展示デモ |
| iREX | 5 | 展示会 |
| 実験室 | 5 | 開発検証 |
| **合計** | **15スキル** | |

**使用時間:** 3Qだけで約1500時間（さくらH200）

### FY26計画

| 環境 | スキル数（想定） | 備考 |
|------|-----------------|------|
| 実現場A（神戸工場等） | 5〜10 | PoC |
| 実現場B（彩都倉庫等） | 5〜10 | PoC |
| 京橋（CXC） | 5 | 展示・デモ |
| ヒューマノイド研究 | 5 | Seeds |
| 横浜 | 3〜5 | TBD |
| **合計** | **25〜40スキル** | |

### コスト試算

**前提:**
- FT単価の試算根拠
 - 1スキルあたりの学習: 約200〜500 GPU時間（A100換算）
 - NC24ads A100: $3.67/h × 50%割引 = 約275円/h
 - 1スキル: 300h × 275円 = **約8万円/スキル**

| シナリオ | スキル数 | FT費用 | 推論・実験費用 | 合計 |
|----------|----------|--------|---------------|------|
| 最小 | 25 | 200万 | 400万 | **600万** |
| 想定 | 35 | 280万 | 560万 | **840万** |
| 最大 | 45 | 360万 | 720万 | **1,080万** |

**採用値:** 月70万円 × 12ヶ月 = **840万円/年**
