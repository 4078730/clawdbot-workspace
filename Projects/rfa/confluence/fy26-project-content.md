# FY26 Robotics Foundation Agent — プロジェクトディスカッション用

---

## 1. プロジェクト概要

**物流・製造現場の「人にしかできない」作業を、AIロボットで自律化する。**

LLM Agent・VLA（Vision-Language-Action）・Robo Syncを統合し、言語指示だけでロボットが状況を理解し、柔軟に動く。FY26は現場PoCを通じて省人化・品質安定・24h運用の実現可能性を検証し、FY27の事業判断に必要なエビデンスを確立する。

### 基本情報

| 項目 | 内容 |
|------|------|
| **種別** | 継続（FY25 → FY26） |
| **継続元** | Robotics Foundation Agent (RFA) |
| **計画年数** | FY24-27の段階計画 / FY26はPhase 2（現場PoC・事業検証） |
| **技術戦略の位置づけ** | Industrial Embodied AI / RoboSync AI Suit Product (RFA Seed) |

### FY25実績

| 項目 | 内容 |
|------|------|
| **リソース** | 67人月 / 7,102万円（経費 約2,554万円、総額 約9,656万円） |
| **主な成果** | VLA+LLM Agent統合アーキテクチャ確立、CXC常設デモ統合、代表タスクの実機PoC達成 |
| **残課題** | Robo Syncスキルブロック化、DataOpsループ確立、現場PoCの定量評価 |

## 2. 目標 / マイルストーン / 成果物

FY26は「現場で回る」ことを最優先とし、FY27以降の事業判断に向けた定量エビデンスを整備する。

### FY26（四半期別）

| Q | 目標 | マイルストーン | 成果物 |
|---|------|----------------|--------|
| **Q1 (4-6月)** | VLAスキルブロック仕様v1を確定し、現場PoC準備を開始 | 仕様合意（Robo Sync側）/ データスキーマ定義 / 事業部とPoC対象合意 | 仕様書v1 / データ契約 / PoC対象合意メモ |
| **Q2 (7-9月)** | 2〜3タスクをAgentが切替実行し、評価指標を確定 | Agent→VLA切替デモ / パイプラインv1稼働 / KPI合意 | デモ動画 / KPI定義（成功率・タクト・人介在） |
| **Q3 (10-12月)** | 現場PoCを開始し、定量評価を取得 | 神戸/彩都でVLA動作検証 / KPI取得 | PoC中間レポート（KPI一次結果） |
| **Q4 (1-3月)** | PoC評価を完了し、事業判断材料を整備 | KPI分析 / 事業仮説レビュー | FY27計画 / 事業検討資料 |

### FY27〜FY28（年次）

| 年度 | 目標 | マイルストーン | 成果物 |
|------|------|----------------|--------|
| **FY27** | 社会実装に向けた本番運用準備 | 現場PoC拡張 / 運用スキーム確立 / 事業部合意 | 本番運用計画 / 事業判断パッケージ |
| **FY28** | パートナー現場での本番運用 | 本番稼働 / 横展開検討 | 導入実績 / 横展開ロードマップ |
