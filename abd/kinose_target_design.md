## RFA全体（95%）— 概要レベルのコメント

- RFAプロジェクトリーダーとして、技術方向性の策定からチーム運営、外部連携まで統括
- FY25はRFAコアスタック（LLM Agent × VLA × Symbol Integration × Robo Sync）の確立をPhase 1目標として推進
- 3層アーキテクチャ（Task Planning Hub / Symbol Hub / Sensorimotor Hub）の設計と参照実装を完了
- FY26事業計画の策定を主導し、「研究→現場PoC」フェーズ移行の準備を完了
- Microsoft協業（Azure MLOps基盤構築）の技術リードとして、ADS完了・Sprint 0開始準備まで推進

## ① 統合PoCシステム（LLM Agent＋RFM）の実装とデモ展示

### 実績

**Q1（4〜6月）: 福岡拠点VLA実証 & 環境整備**

- 福岡拠点（SDL FUK）にてVLA（UR5e）によるビンピッキング検証を立ち上げ・推進
  - 月1回ペースで現地に通い、Q2末（責任者報告会）まで継続。報告会を節目にピボット
  - RoboSync DaVinci上のルールベース制御起動 → データ収集 → ROSbag→RLDS変換 → Azure上へアップロード → ファインチューニング → デプロイ・評価の一連工程をほぼ一人で実施
  - 性能向上には至らなかったが、MS協業のMLOps/DataOps、VLA→RoboSync組込パイプラインの先行プロトタイプ検証として大きな成果
  - ROSベース・UR5eベースでのVLA学習の技術的実現性と、数値評価による意義の証明を確認
- Koch → SO-101等のオープンソースアームをIdea Forgeにセットアップし、デモ環境を段階的に整備（Q2頃、小栗さん協力）
- W&B（Weights & Biases）の契約対応

**Q2（7〜9月）: iREX企画 & 実験・デモ環境拡充**

- 7月にMI本部とのディスカッションを実施（コネクト×ホールディングス連携）
- iREX 2025に向けた企画を始動
  - ホールディングス側の力制御開発と、LLM Agent / RoboSync向けVLAの統合を技術的に詰めるフェーズへ
- 6月末：ドイツ・ミュンヘン「Automatica」展示会を1週間視察 → 帰国後、ロボット業界の現状をコネクト内部に共有
- 福岡に2〜3日泊まり込みで実験を実施。テレオペなしでのデプロイ・ロールアウトまでを一人でやり切った
- Trossen AI（旧Aloha）AIステーション関連ロボットを購入 → 大型ハードウェアでの本格実験環境を確保
- VLA学習のためにさくらHPCを導入（小栗さんがセットアップ・導入ガイドラインを推進）
- 7月「ファミリーデー」にてSO-101テレオペデバイスを使ったロボット体験会を企画・実施（社員の子ども向け、お菓子詰め体験）→ 社内交流・技術への親しみ創出に貢献
- CXC祝祭広場デモ展示（8月末〜9月）
  - 小栗さんがSO-101でのデモ環境を構築し実機を稼働
  - 一般来場者が集まる天神の広場で、VLA自律型AIロボットが安定動作する研究成果をアウトプット
  - 技術者付きっきりでない場での展示として、パナソニックの最先端技術への取り組みと事業化姿勢をアピール
  - 小栗さん・黒川さん・Zhuさんがタイトスケジュールの中で設営を完遂

**Q3（10〜12月）: iREX展示 & 対外発信の飛躍**

- 9月上旬：CXCリニューアル展示を完了（CXC祝祭広場でのVLAデモの発展版）
- 9月19日：責任者報告会を完了（福岡VLA実証の成果報告を含む）
- 9月〜：池内先生との定期ディスカッションを開始（上野さんからの紹介。研究方向性の壁打ち相手として）
- 10月：スウォンジー大学とのミーティング実施 — テーマディスカッション＋Zetes幹部メンバーへのデモ
- 10月23日：ファンケル物流会社を訪問（黒川さん企画）→ 物流・梱包へのユースケース選定意識が深化
- IndexVLAの実装を進め、実験可能な状態まで到達（ただしデモの入れ替わりが多く実験が滞る課題あり）
- 黒川さん主体で技術シンポジウムへの投稿を実施（RSJベースの内容、ポスター・論文の作成を指導）
- コネクトハッカソン（3日間）を開催 — Connect内外メンバーとの協働でエンゲージメント向上に貢献
- iREX 2025（11/30〜）の展示を統括
  - 「シェーバーを梱包する」タスクを選定。VLAの価値訴求の方向性を松山さん・ロボット部・PhD等とディスカッションし資料化
  - 法務処理・業務手続きのリード、データ収集・ロボット調整・当番スケジュール管理まで統括
  - 4スキル連続実行による梱包デモを実現。非定型物体（袋物・不定形物）対応の実機動作を対外発信
  - **パナソニックとして初のVLAアウトプット** — プレスにも掲載、「VLA パナソニック」で検索可能に
  - 認知拡大により、その後のコネクション形成・コミュニケーション促進に直結
  - 小栗さんがロボット配送手配等のロジを尽力

**Q4（1〜3月）: CXCデモ統合 & FY26準備**

- CXCデモの企画・設計・実装を統括
  - ギフト梱包・返品対応・在庫確認の3シナリオ（9バリアント）を定義
  - VLA 5スキル（箱開け/閉め/アイテム投入/取出/リボン付け）+ ルールベース2スキルの構成を設計
  - LLM Agent（Claude Haiku 4.5）+ VLM（Gemini 2.0 Flash）+ VLA（LeRobot）の統合アーキテクチャを構築
  - 黒川さん（VLM失敗成功判定）、村上さん（プランニング）への開発分担を設計・管理
- デモ動画・技術紹介資料を制作し、Blue Yonderドイツメンバー等の社外来訪者向けにIdea Forgeでプレゼンテーション実施
- CXC常設展示リニューアル（3月完成予定）を推進中
  - 「VLA単体デモ」→「VLA + LLM Agentによるマルチタスク長期タスクデモ」へのアップデート
