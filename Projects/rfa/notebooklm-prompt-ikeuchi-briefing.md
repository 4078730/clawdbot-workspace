### スライド1: タイトル + サマリー

**池内先生ディスカッション ブリーフィング**

池内先生との連携で得られた**3つの知見**：

1. **VLAの学習単位** → GMR（Grasp-Manipulation-Release）
2. **スキルの再利用** → 物理特性で分類、新タスク展開コスト削減
3. **継続学習サイクル** → 「使いながら賢くなる」が差別化

### スライド2: 課題と解決策

**【図】VLAの粒度問題と解決策**

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   粒度が大きすぎる        適切な粒度           粒度が小さすぎる   │
│   ┌─────────┐          ┌─────────┐          ┌───┐ ┌───┐     │
│   │ 梱包全体 │    →     │   GMR   │    ←     │Pick│ │Place│   │
│   │         │          │ 把持→操作 │          └───┘ └───┘     │
│   │ 複雑すぎ │          │ →解放   │          組み合わせ爆発   │
│   └─────────┘          └─────────┘                          │
│       ✗                    ✓                    ✗           │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

**GMR = Grasp（把持）→ Manipulation（操作）→ Release（解放）**
- 池内先生の研究：接触状態遷移で操作を定義
- 「ちょうどいい粒度」：意味のあるまとまり、再利用可能
