## Scene 001: 進撃の巨人 — エレン・イェーガー「この裏切り者がぁぁぁ!!」

### 原作シーン詳細
- **出典:** Season 2 第31話「戦士」〜第32話
- **状況:** ウォール・ローゼ上。エレンの目の前で、同期であり戦友だったライナー・ブラウンが突然「俺が鎧の巨人で、こいつ（ベルトルト）が超大型巨人だ」と告白する。エレンは最初「お前…疲れてるんだよ」と信じようとしないが、ライナーが実力行使に出た瞬間、裏切りを確信。怒りと絶望で顔が歪み、「この裏切り者がぁぁぁ!!」と絶叫しながら自らの手を噛み、巨人化する。
- **場所:** ウォール・ローゼの城壁の上。石造りの壁面、曇天、風が強い。背景に壁外の広大な平原。
- **時間帯:** 昼間（曇り空、薄暗い光）
- **エレンの服装:** 調査兵団の制服（茶色のジャケット、白いシャツ、立体機動装置のベルト）、調査兵団マント（自由の翼エンブレム）

### ファーストフレーム画像の要件
- エレン正面バストアップ〜ミディアムショット
- 調査兵団制服着用
- 表情: 怒りが込み上がる直前の苦悩・動揺（まだ叫んでいない）
- 背景: 石造りの城壁の上、曇り空

### プロダクション

**尺:** 10秒
**アスペクト比:** 9:16
**音声:** ON

---

**CUT 1（0〜3秒）— 絶望から怒りへ**

```
エレンの表情がゆっくりと苦悩から激しい怒りに変わる。目が見開かれ、歯を食いしばり、顔が震える。風がマントと髪を激しくなびかせる。カメラはゆっくりドリーインで顔に寄っていく。正面バストアップ。曇天の城壁の上、背景に壁外の荒野。低い風の音と不穏なドローン音。
```

**CUT 2（3〜7秒）— 絶叫と巨人化の決意**

```
エレンが天を仰いで「裏切り者がぁぁ!!」と絶叫する。涙が飛び散り、叫びで首の筋が浮き上がる。右手をゆっくり口元に持っていく。決意の込もった目でカメラを睨みつける。ロングからバストアップに急速ズーム。城壁の上に強風が吹き荒れる。絶叫の反響音、風切り音。
```

**CUT 3（7〜10秒）— 手を噛む・巨人化の閃光**

```
エレンが自分の手に噛みつく瞬間、口元のクローズアップ。歯が皮膚に食い込み、血が滴る。直後、体から黄金の稲妻が爆発的に走り、蒸気が噴き出す。カメラが衝撃で後方に引かれる。見上げアングルでエレンのシルエットが巨大化していく影。雷鳴のような巨人化の轟音、蒸気の噴出音。
```

## Scene 002: ワンピース — モンキー・D・ルフィ「ギア5 ニカ覚醒」

### 原作シーン詳細
- **出典:** ワノ国編 第1071話（アニメ）/ 漫画第1044話「解放の戦士」
- **状況:** 鬼ヶ島の屋上。カイドウとの死闘でルフィが一度敗北し心臓が止まる。しかし「ゴムゴムの実」の真の姿——ヒトヒトの実 幻獣種 モデル:ニカが覚醒。ルフィの髪と服が白く変わり、眉毛がくるりと巻き、瞳が赤く光る。体からは蒸気が立ち上り、心臓が「ドンドットット」と太鼓のようなリズムを刻む。ルフィはケタケタと笑い始め、これまでにない自由な動きを見せる。
- **場所:** 鬼ヶ島 屋上（夜空、満月、瓦礫が散乱、炎が燃える廃墟的な戦場）
- **時間帯:** 深夜（月明かりが主光源、炎のオレンジが補助光）
- **ルフィの外見（覚醒後）:** 白い髪（逆立ち、炎のようにうねる）、白い服、赤い瞳、くるりと巻いた眉毛、体から白い蒸気、常に笑っている

### ファーストフレーム画像の要件
- ルフィ（覚醒直前 or 覚醒後）のミディアムショット
- 覚醒後を使う場合: 白髪、白い服、赤い瞳、笑顔
- 背景: 夜空、月、鬼ヶ島の廃墟的な屋上

### プロダクション

**尺:** 10秒
**アスペクト比:** 9:16
**音声:** ON

---

**CUT 1（0〜3秒）— 心臓の鼓動と覚醒の兆し**

```
ルフィが仰向けに倒れている。沈黙の中、心臓が「ドン…ドットット」と太鼓のようなリズムを刻み始める。体から白い蒸気がゆっくりと立ち上る。髪が白く変色していく。見下ろしアングルから始まり、カメラがゆっくり回り込む。夜空と満月の背景。太鼓のリズム音だけが響く。
```

**CUT 2（3〜7秒）— 立ち上がり・笑い始める**

```
ルフィがむくりと起き上がる。白く変わった髪が炎のように逆立ち、蒸気が体を包む。口元がゆっくりと笑みの形に変わり、やがて堪えきれないようにケタケタと笑い出す。赤く光る瞳が見開かれ、歯を見せた満面の笑み。カメラが正面からゆっくりティルトアップ。月明かりと炎に照らされた戦場。太鼓のリズムが加速、ルフィの笑い声。
```

**CUT 3（7〜10秒）— ニカの力を解放**

```
ルフィが拳を握りしめ、天に向かって掲げる。白い蒸気が爆発的に噴き出し、地面が溶けるように変形する。体がゴムのように伸縮しながら力強く構える。満月を背にしたシルエットショット。白い髪と蒸気が風になびく。見上げアングル。太鼓のクライマックス、地面が割れる音、解放感のある風の音。
```

## Scene 003: 鬼滅の刃 — 我妻善逸「雷の呼吸 壱ノ型 霹靂一閃・六連」

### 原作シーン詳細
- **出典:** 立志編 第17話「ひとつのことを極め抜け」（那田蜘蛛山編）
- **状況:** 那田蜘蛛山にて、善逸は蜘蛛の鬼（兄）に毒を注入され、体が蜘蛛に変わり始めている。恐怖で泣き叫んでいたが、あまりの恐怖に気を失い眠りに落ちる。眠った瞬間、善逸の人格が一変——怯えは消え、研ぎ澄まされた剣士の目になる。師匠・桑島慈悟郎の「ひとつのことしかできないならそれを極め抜け」という言葉を思い出し、唯一使える技「雷の呼吸 壱ノ型 霹靂一閃」を六連続で放ち、鬼の首を一瞬で斬り落とす。
- **場所:** 那田蜘蛛山の森の中。巨大な蜘蛛の巣が張り巡らされた暗い森。木々の隙間から月明かりが差す。
- **時間帯:** 夜（月明かりがスポットライトのように差し込む）
- **善逸の外見:** 鬼殺隊の隊服（黒い詰襟）に黄色い羽織。金髪のボブカット。目を閉じている（眠っている）。体の一部が蜘蛛化し始めている（片腕が変色）。刀を構えた「居合」の姿勢。

### ファーストフレーム画像の要件
- 善逸が刀を鞘に収めた居合の構え（目を閉じている）
- 鬼殺隊の隊服＋黄色い羽織
- 背景: 暗い森、蜘蛛の巣、月明かりが上から差し込む
- 表情: 眠っている（穏やかだが研ぎ澄まされた雰囲気）

### プロダクション

**尺:** 10秒
**アスペクト比:** 9:16
**音声:** ON

---

**CUT 1（0〜4秒）— 眠りの静寂・居合の構え**

```
善逸が目を閉じたまま居合の構えで静止している。風が黄色い羽織と金髪を僅かに揺らす。月明かりが上から一筋差し込み、善逸だけを照らす。周囲は蜘蛛の巣が張り巡らされた暗い森。静寂の中、呼吸音だけが聞こえる。カメラはゆっくりドリーインで全身からバストアップに寄っていく。虫の声、風の音、そして微かな雷鳴。
```

**CUT 2（4〜6秒）— 雷の呼吸発動・抜刀の瞬間**

```
善逸の足元に黄金の雷紋が走る。一瞬の溜めの後、目にも留まらぬ速さで抜刀。雷のエフェクトが体を包み、黄金の軌跡が画面を切り裂く。スローモーションで刀が鞘から抜ける瞬間のクローズアップ。月明かりが刀身に反射。雷鳴のような抜刀音、空気を切り裂く鋭い音。
```

**CUT 3（6〜10秒）— 霹靂一閃・六連の残像**

```
善逸が雷速で六方向に斬撃を繰り出す。黄金の雷の軌跡が六本の線となって暗闇を引き裂く。善逸の残像が複数重なり合い、最後に元の位置で刀を鞘に納める。納刀のカチンという音と同時に、蜘蛛の巣が一斉に切断されて崩れ落ちる。見上げアングルで月を背にした善逸のシルエット。連続する雷鳴、納刀音、蜘蛛の巣が崩れる音、最後に静寂。
```
