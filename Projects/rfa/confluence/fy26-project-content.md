## 1. プロジェクト内容

### 概要

**【生み出される技術】**
- LLM Agent + VLA（Vision-Language-Action）+ Robo Sync の統合基盤
- 現場データを学習資産に変換するDataOpsパイプライン

**【何ができるようになるか】**
- 言語指示だけでロボットが状況を理解し、非定型作業を自律実行
- 失敗時のリカバリー、タスク再計画を自動化
- 使うほど賢くなる継続学習サイクル

**【Industrial Embodied AIへの事業貢献】**
- 物流・製造現場の属人作業を自律化 → 省人化・品質安定・24h運用
- Robo Sync導入現場からのデータ蓄積 → 競争優位の源泉
- FY27以降の事業化判断に必要なPoCエビデンスを確立

**【仮説の妥当性】**
- 市場：SCM領域の人手不足は構造的課題
- 競合優位：Deploy-Time Scaling戦略によるデータ蓄積
- 技術機会：VLA/LLMの急速な発展

> 📄 詳細：[FY26 RFA 事業計画書](https://panasonic-connect.atlassian.net/wiki/spaces/CT1/pages/2484830627)
