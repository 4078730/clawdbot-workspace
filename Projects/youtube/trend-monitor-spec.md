## 1. 背景と課題

### チャンネル概要
- **チャンネル名:** AnimeLiveAction
- **コンセプト:** アニメの名シーンをAI実写再現（Dreamina / Seedance 2.0）
- **フォーマット:** 縦型 9:16、一人のキャラクターにフォーカス
- **制作ツール:** Dreamina（Seedance 2.0）、中国語スタイルプリセット + 日本語脚本プロンプト

### 課題
SNSのトレンドは日々変化しており：
- **AIコンテンツのトレンド** — バズるAI動画のスタイル・手法・ツールが毎日変わる
- **アニメのトレンド** — 話題の作品・キャラ・シーンが放送や公開と連動して動く
- これらを手動で追うと情報収集に時間がかかり、タイミングを逃す
- 「今何を作るべきか」の判断が直感頼みになっている

---

## 2. 目的とゴール

**リアルタイムのトレンド情報を自動収集・整理し、「今作るべきコンテンツ」の判断を高速化する。**

| ゴール | 指標 |
|--------|------|
| トレンド検知の遅延をなくす | 1日以内にキャッチ |
| 判断に必要な情報が整理された状態で届く | デイリーブリーフィング |
| 制作判断を支援する | レコメンドと優先度付き |

---

## 3. クリエイティブ方針

### コンテンツの方向性
- ✅ **名シーン実写再現**（一人のキャラに焦点）
- ✅ **撮影現場の舞台裏**（有名アニメの制作現場のような世界観）— 実績あり、チャンネルの強み
- ✅ **リアルと溶け合った世界観**（HOKUSに通じるスタイル）
- ✅ 縦型動画（9:16）→ 一人構図が映える
- ✅ 同一作品内のIFバトル（品のある範囲で）
- ✅ 実写化のクオリティとリスペクト
- ❌ 異作品クロスオーバー
- ❌ アニメ画像をそのまま動かすだけ

### 参考: HOKUS（ホコス）
- リアルとCGが溶け合った映像体験
- 「撮影現場の裏側」は現実世界にアニメキャラが存在するかのような没入感
- このスタイルはトレンドに合致しており、差別化にもなる

## 4. モニタリング対象

### ソース優先度

| 優先度 | プラットフォーム | 理由 |
|--------|----------------|------|
| 🔴 高 | **X (Twitter)** | AIクリエイター＋アニメ公式の両方が集まる。リアルタイム性最強 |
| 🔴 高 | **YouTube / Shorts** | 競合チャンネルの動向、再生数データが取れる |
| 🔴 高 | **TikTok** | 短尺動画のトレンド発信源。音源トレンドも重要 |
| 🟡 中 | **Instagram** | Reels中心に良質なAIコンテンツが増加中 |
| 🟢 低 | **Reddit** | r/StableDiffusion, r/aivideo — ツール・ワークフロー情報 |
| 🟢 低 | **中国系**（Douyin, 小紅書, Bilibili） | Dreamina/Seedance活用例、中国発トレンド |

### モニタリング軸

#### A. AIコンテンツトレンド
| 取得したい情報 | ソース |
|--------------|--------|
| バズったAI動画（再生数・エンゲージメント） | X, YouTube, TikTok, Instagram |
| 使用ツール・新モデル発表 | X, Reddit |
| プロンプト手法・ワークフロー | X, Reddit, 中国系 |
| 競合チャンネルの新作・再生数推移 | YouTube |

#### B. アニメトレンド
| 取得したい情報 | ソース |
|--------------|--------|
| 話題のシーン・キャラ人気 | X, TikTok |
| 放送直後の反応・実況 | X |
| 新作・続編の放送スケジュール | MyAnimeList, AniList |
| 検索ボリュームの急上昇 | Google Trends |

#### C. クロス分析（最も価値が高い）
> **「今バズっているアニメ」×「AI実写化で映えるシーン」**の交差点を見つける

## 5. 機能要件

### 5.1 タグ＆フィルタシステム

各トレンド項目にタグを付与し、限定検索を可能にする。

**タグカテゴリ:**

| カテゴリ | タグ例 |
|---------|--------|
| ソース | `X` `YouTube` `TikTok` `Instagram` `Reddit` `CN` |
| コンテンツ種別 | `AI動画` `AI画像` `アニメ話題` `ツール情報` `放送予定` |
| 作品 | `呪術廻戦` `チェンソーマン` `推しの子` `ワンピース` ... |
| スタイル | `実写化` `変身シーン` `バトル` `日常` `舞台裏` |
| 緊急度 | `🔥トレンド急上昇` `📈じわ上昇` `📅予定イベント` |

**フィルタ操作:**
- チェックマークで複数タグをON/OFF
- 例: `[✅ X] [✅ TikTok] [✅ 呪術廻戦]` → Xとイクトクの呪術廻戦関連だけ表示

### 5.2 デイリーブリーフィング

毎朝自動生成される日次レポート:

```
📊 トレンドブリーフィング (2026-02-13)

🔥 AIトレンド
━━━━━━━━━━━━━━━━━━━━━━━━
[X] @creator のワンピース実写 → 120万再生（Dreamina, マルチショット）
[TikTok] #AIanime「変身シーン」急上昇（+340%）
[Instagram] Reelsで「舞台裏」系AIコンテンツがバズ傾向

📺 アニメトレンド
━━━━━━━━━━━━━━━━━━━━━━━━
[X] 呪術廻戦 最終話放送 → トレンド1位（12h継続）
[Google] チェンソーマン 映画公開日発表 → 検索+400%

🎯 制作レコメンド
━━━━━━━━━━━━━━━━━━━━━━━━

2. 🟡 デンジ変身シーン（映画発表で検索急上昇）
```

### 5.3 オンデマンド実行

手動でリクエストして即時更新する機能:
- Discord上で `!trend` や特定コマンドで実行
- 特定ソースや作品に絞った検索も可能

### 技術選択

| 選択肢 | メリット | デメリット |
|--------|---------|----------|
| **静的HTML + JS（推奨）** | 軽量、デプロイ不要、OpenClaw canvasで表示可 | 動的更新はJSON再生成 |
| Streamlit | Python直結、グラフ簡単 | サーバー常駐が必要 |
| Notion | 既存ツール活用 | API制約、カスタマイズ性低い |

**推奨: 静的HTML + Chart.js** → JSONデータを毎朝生成、HTMLがそれを読み込んで表示

## 7. 運用フロー

### 自動運用（毎朝）

```
[07:00 JST] cronジョブ起動
    ↓
データ収集スクリプト実行（各API/スクレイピング）
    ↓
LLMで要約・タグ付け・レコメンド生成
    ↓
JSONデータ更新 → ダッシュボードHTML自動更新
    ↓
デイリーブリーフィングをDiscordに送信
```

### 手動運用（オンデマンド）

```
konuki: 「今のトレンド教えて」 or !trend
    ↓
即時データ収集 → LLM分析 → 結果をDiscordに返信
```
