## プロジェクト概要

**RFA CXC Demo** - VLA（Vision-Language-Action）ポリシーを活用したロボット制御デモシステム。CXC常設デモとして、非技術者でも運用できる形でギフト梱包と返品対応を実演。

### ステータス
- **完了度**: 98%
- **フェーズ**: 実装完了、実機テスト待ち
- **最終更新**: 2026-01-13

---

## 展示の目的・メッセージ

### 大目標
- サプライチェーンの強靭化・効率化
- AI制御ロボットによる労働力不足対応・生産性向上

### コアコンセプト
「現場で"失敗できるAI"を育てる」
- CXCを"デモ展示の場"から"現場AIの育成・検証拠点"へ
- 「使えることを目指す」「導入ハードルを下げる」「データで差別化する」

### コネクトの強み・差別化
- **Robo Sync**: 現場知見の蓄積、現場で動くデータを素早く収集
- **スピード**: 翌日にはAIモデルが現場で稼働可能
- **ハード選択の自由**: パナソニック製に縛られない
- **データループ**: 失敗→学習→改善が回る

## 技術構成

### アーキテクチャ
| 層 | 役割 | 担当 |
|---|------|------|
| 意図推定・対話 | 言語理解、確認、計画提示 | LLM Agent (Claude Haiku 4.5) |
| 状態認識 | 箱の開閉、アイテム位置検出 | VLM (Gemini 2.0 Flash) |
| 器用な作業 | Put X in box、袋入れ等 | VLA Policy (LeRobot) |
| 安定した工程 | ホームポジション、待機 | ルールベース（Robo Sync連携予定） |

### 技術スタック
- **Backend**: Python 3.10+, FastAPI, WebSocket, asyncio
- **Frontend**: React 18+, TypeScript, Vite
- **AI/ML**: Google Gemini 2.0 Flash (VLM), Anthropic Claude Haiku 4.5 (LLM Planner)
- **Robot**: LeRobot

## シナリオ定義（3種類、9バリアント）

### 1. 🎁 ギフト梱包 (gift_packing)
アイテムを選んで箱に梱包

**バリアント:**
| ID | 名前 | 推奨アイテム |
|----|------|-------------|
| kids | 子ども向け | チョコスティック、メモ帳、ボールペン |
| male | 男性向け | メモ帳、ボールペン、ポケットティッシュ |
| female | 女性向け | ハンドクリーム、メモ帳、チョコスティック |
| business | ビジネス向け | メモ帳、ボールペン |
| casual | カジュアル | チョコスティック、ポケットティッシュ |
| premium | プレミアム | ハンドクリーム、メモ帳、ボールペン、チョコスティック |

**ワークフロー:** open_box → pick_place_loop → close_box

### 2. 📦 返品対応 (return_handling)
返品された商品を確認し、正しい商品と入れ替え

**バリアント:**
| ID | 名前 | 推奨アイテム |
|----|------|-------------|
| default | 標準 | メモ帳、ボールペン |
| wrong_item | 誤品交換 | ハンドクリーム、チョコスティック |

**ワークフロー:** open_box → remove_wrong_items → add_missing_items → close_box

### 3. 📋 在庫確認 (inventory_check)
作業台のアイテムを確認してレポート

**ワークフロー:** scan_items → report

## スキル定義（5 VLA + 2 ルールベース）

### VLAスキル
| ID | 名前 | Port | 説明 |
|----|------|------|------|
| vla_open_box | 箱を開ける | 18001 | 箱のフタを開ける |
| vla_close_box | 箱を閉める | 18001 | 箱のフタを閉める |
| vla_put_into_box | {item}を箱に入れる | 18002 | アイテムを把持して箱に入れる |
| vla_take_from_box | {item}を箱から出す | 18003 | アイテムを箱から取り出してアイテムエリアに戻す |
| vla_attach_ribbon | リボンを付ける | 18004 | リボンを箱のフタに付ける |

### ルールベーススキル
| ID | 名前 | 説明 |
|----|------|------|
| rule_home_pose | ホームポジション | ロボットをホームポジションに戻す |
| rule_wait | 待機 | 指定時間待機 |
