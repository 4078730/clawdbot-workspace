# キミチカ 〜キミの近くに座りたい〜

## 概要

**サークル:** Re-one
- サイト: <https://reone-boardgame.github.io/index.html>
- Google Sites: <https://sites.google.com/view/re-one/>

**ジャンル:** 青春×席替えバトルゲーム

## ゲーム情報

- **プレイ人数:** 3-6人
- **対象年齢:** 10歳〜
- **プレイ時間:** 40分〜

## コンセプト

放課後の「理想の席替え」談義。好きな子の隣を狙いつつ、本音は隠し通す。バレたら噂を広められてからかわれる！

**ゲームの目的:**
- 好きな人の近くに自席を配置して「席次ボーナス」獲得
- 他プレイヤーの好きな人を当てて「推理ボーナス」獲得
- 3学年（ラウンド）で最も好感度チップを集めた人が勝利

## 内容物

- 座席カード: 24枚（8部活 × 各3枚）
- 好きな人カード: 8枚
- 自席カード: 12枚（6色 × 各2枚）
- 好感度チップ: 1pt × 30枚、5pt × 10枚
- 中央マーカー: 1枚
- ルール説明書: 1枚
