5. Task Process Detail

α Pick Cycle · β Full Workflow

α Version Pick Cycle — S0 to S10

Pick Cycle — Step Summary

StepActionModuleTarget Time
S0Move to home positionRule-based2s
S1Detect bin arrivalRule-based1s
S2Recognize parts in binVLM2s
S3Select pick target + grip strategyPMS1s
S4–S6Approach → Grasp → LiftVLA Policy10s
S7Transport (bin → table)VLA / Rule3s
S8Place on work tableVLA Policy3s
S9Verify placementVLM2s
S10Return to homeRule-based2s
Total — 1 Pick Cycle~26s

Human worker: ~1–3 min/order. α target: same order of magnitude, prioritizing reliability over speed.

β Version — Full Outbound Cycle [1]–[8]

[Step]Action
[1]Barcode scan (delivery slip)
[2]Monitor operation (bin call)
[3]Open cardboard in bin (if needed)
[4]Pick parts (α extension)
[5]Return remainder to bin
[6]Individual packaging (bag + slip + seal)
[7]Place on wagon
[8]Next order → repeat
Key addition vs α: Task Planning Hub (LLM Agent) orchestrates [1]–[8] via Behavior Tree skill switching.

Hardware upgrade: Dual UR5e (required for packaging tasks requiring bimanual coordination)

Priority: "Make it work reliably" before optimizing cycle time.
