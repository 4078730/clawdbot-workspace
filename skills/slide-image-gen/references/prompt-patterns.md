# Prompt Patterns by Figure Type

## 1. Flowchart / Process Flow

Use for: workflows, decision trees, step-by-step processes

```
Flowchart diagram on white background (#FFFFFF).
Rectangular boxes in navy (#000E4E) with white text for process steps.
Diamond shapes in cyan (#00B7F1) with navy text for decision nodes.
Arrows in navy. Clean, left-to-right or top-to-bottom flow.
Academic style, no shadows or gradients.

Steps: [LIST EACH STEP]
Decision points: [LIST EACH DECISION AND BRANCHES]
Label all connectors. Minimal whitespace between nodes.
```

---

## 2. System / Architecture Diagram

Use for: software architecture, data pipelines, layered systems

```
System architecture diagram on white background (#FFFFFF).
Layered horizontal bands: navy (#000E4E) for layer headers with white text.
Component boxes inside layers: white fill with navy border, navy text.
Arrows and data flow lines in cyan (#00B7F1).
Clean grid layout, consistent box sizing.

Layers (top to bottom): [LAYER NAMES]
Components per layer: [COMPONENT LIST]
Data flow direction: [DESCRIBE CONNECTIONS]
Label all arrows with action/data type.
```

---

## 3. Comparison / Matrix

Use for: option comparison, before/after, feature matrix

```
Comparison table/matrix on white background (#FFFFFF).
Header row in navy (#000E4E) with white text.
Alternating rows: white and very light gray (#F5F5F5).
Highlight cells in cyan (#00B7F1) for key differentiators.
Clean sans-serif font, academic style.

Columns: [COLUMN HEADERS]
Rows: [ROW ITEMS]
Highlight these cells as best/recommended: [SPECIFY]
```

---

## 4. Timeline / Roadmap

Use for: project schedule, milestones, phase planning

```
Horizontal timeline diagram on white background (#FFFFFF).
Timeline bar in navy (#000E4E).
Milestone markers as cyan (#00B7F1) circles with navy labels above.
Phase blocks as navy-bordered rectangles below the timeline.
Clean, academic, left-to-right progression.

Time range: [START] to [END]
Milestones: [DATE — LABEL pairs]
Phases: [PHASE NAME with date range]
```

---

## 5. Concept / Relationship Diagram

Use for: framework visualization, relationships between concepts, hierarchies

```
Concept diagram on white background (#FFFFFF).
Central concept: large navy (#000E4E) circle/box with white text.
Related concepts: smaller white boxes with navy borders and navy text.
Connecting lines in cyan (#00B7F1) with relationship labels.
Radial or hierarchical layout. Minimal, academic style.

Central concept: [NAME]
Related concepts: [LIST]
Relationships (edges): [CONCEPT A — RELATIONSHIP — CONCEPT B]
```

---

## 6. Data Chart / Statistical Figure

Use for: bar charts, line graphs, pie charts with data

```
[CHART TYPE] chart on white background (#FFFFFF).
Primary data series in navy (#000E4E).
Secondary/comparison data series in cyan (#00B7F1).
Axis lines and gridlines in light gray (#CCCCCC).
No chart border. Clean, publication-quality academic style.

Data: [DESCRIBE DATA OR PROVIDE VALUES]
X-axis label: [LABEL]
Y-axis label: [LABEL]
Legend: [SERIES NAMES]
Title: [CHART TITLE]
```

---

## 8. 3D Isometric Layout / Scene

Use for: physical workstation layouts, equipment placement, spatial relationships

```
3D isometric illustration on white background (#FFFFFF).
Clean technical isometric style — professional documentation quality.
Slight depth shading, not photorealistic. Academic feel.

Color palette:
- Surfaces/floor: light gray (#F0F0F0) with isometric depth shading
- Key equipment: navy (#000E4E) body, cyan (#00B7F1) joint/cable highlights
- Work surfaces: muted [COLOR] fill with dark edges
- Labels: navy sans-serif with leader lines

Scene elements:
[LIST EACH OBJECT WITH POSITION: front-left / center / back-right / etc.]

Label each element with leader lines:
[ELEMENT NAME (spec/dimension)] — pointing to [object]

Title top-left: '[SCENE NAME]' navy bold
Subtitle: '3D Isometric View ([source])' small navy
Note bottom: '* Subject to on-site verification' small italic navy
```

---

## 7. Venn / Overlap Diagram

Use for: capability intersection, strategic positioning

```
Venn diagram on white background (#FFFFFF).
Circle A: semi-transparent navy (#000E4E, 60% opacity).
Circle B: semi-transparent cyan (#00B7F1, 60% opacity).
Overlap region: blended color with white label.
Clean, academic style. Labels outside circles in navy.

Circle A label: [NAME]
Circle B label: [NAME]
Overlap label: [INTERSECTION CONCEPT]
Key points inside each region: [LIST]
```
