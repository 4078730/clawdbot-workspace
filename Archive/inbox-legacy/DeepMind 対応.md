# Robotics Trusted Tester Agreement
## ロボティクス・トラステッドテスター契約

---

## 契約概要

本契約（以下「本契約」）は、以下に定義される**有効日（Effective Date）をもって、Google LLC（以下「Google」）と、本契約に同意する個人または法人（以下「お客様」）との間で締結されます。

---
## 1. 定義

| 用語                   | 定義                                                                          |
| -------------------- | --------------------------------------------------------------------------- |
| **1.1 API**          | Google が提供する Robotics API、関連するソフトウェア開発キット（SDK）、ドキュメント、サンプルコード、その他関連資料を含みます。 |
| **1.2 フィードバック**      | お客様が Google に提供する、API、モデル、ドキュメント、またはプログラムに関する提案、コメント、改善案、修正案、意見等を指します。      |
| **1.3 Google コンテンツ** | Google が本プログラムに関連して提供する、すべての API、モデル、ソフトウェア、ドキュメント、資料、情報等を含みます。             |
| **1.4 機密情報**         | 一方当事者が他方当事者に対して開示する、非公開情報、技術情報、事業情報、製品情報、ロードマップ、ノウハウ等を指します。                 |
| **1.5 モデル**          | Google が提供するロボティクス向け AI モデル（Gemini Robotics、VLA 等を含みますが、これらに限定されません）。       |
| **1.6 プログラム**        | Google が提供する Robotics Trusted Tester Program を指します。                         |

## 2. プログラムへの参加
### 2.1 目的
本プログラムの目的は、Google のロボティクス関連モデルおよび API を、**研究・評価・検証目的**で限定的に提供し、フィードバックを得ることです。
### 2.2 使用制限
- お客様は、Google コンテンツを **研究・評価・非商用目的にのみ** 使用することに同意します。
- **商用利用、第三者への提供、再配布は禁止** されます。
---
## 3. ライセンスの付与
### 3.1 Google からお客様へのライセンス
Google はお客様に対し、以下の条件の下、**非独占的、譲渡不可、サブライセンス不可、取消可能な限定ライセンス**を付与します。
- API およびモデルを、研究・評価目的で使用すること
- Google コンテンツを改変しないこと（Google が明示的に許可した場合を除く）

# Robotics Trusted Tester Agreement 解説（要点整理）

## 0. この契約の位置づけ（最重要）

- これは **「商用契約」ではない**
    
- **研究・実験用途限定の“先行テスト参加契約”**
    
- 目的は：
    
    - Googleが **モデルを改善するための実地フィードバックを得る**
        
    - 参加者（Panasonic側）が **未公開・先行モデルにアクセスできる**
        

👉 **「共同研究の入口」**にあたる契約

---

## 1. Trusted Tester Program とは何か

### ✔ できること

- Gemini Robotics / ER / 将来の VLA などの  
    **未公開・未完成プロダクト（Test Products）**を使用できる
    
- API・ソフトウェア・場合によってはハードウェアも対象
    
- **計算資源は Google が負担**
    
- フィードバックを通じてモデル進化に関与できる
    

### ❌ できないこと

- **本番利用（Production use）は禁止**
    
- 顧客向け展開・商用利用は不可
    
- 安全クリティカル用途は禁止
    

---

## 2. データに関する超重要ポイント（要注意）

### 2.1 データの扱い（核心）

契約上、データは以下に分かれます：

|種類|扱い|
|---|---|
|既存データ（Panasonicが元々持っている）|非独占ライセンス|
|**新規収集データ（Test中に取得）**|**Googleが独占的に利用可能**|
|Model Outputs（モデルの出力）|**すべてGoogleに帰属**|

🔴 **特に重要**

> Newly Collected Data は  
> **Googleに「独占・永続・世界的・サブライセンス可能」な権利を付与**

つまり：

- テスト中に取った
    
    - ロボット動作データ
        
    - 画像・動画
        
    - 行動ログ  
        → **Googleが自由に使える**
        

これは **研究目的の代わりに差し出す対価**と考えるのが正しいです。

---

## 3. 個人データ（Personal Data）に関する制約

### 原則

- **人が映る／声が入る可能性がある場合**
    
    - 事前の **インフォームド・コンセント必須**
        
    - 書面署名が必要
        
- 顔ぼかし等の対策が義務
    

👉 工場・倉庫での実証では  
**「人が映らない構成」か「厳密な同意管理」必須**

---

## 4. 知的財産（IP）の整理

### Panasonic側が保持するもの

- 自社で開発した **周辺ソフトウェア**
    
- ロボット制御コード（Test Productを含まないもの）
    

### Google側に帰属するもの

- Test Products 本体
    
- Model Outputs
    
- Feedback（提案・改善案含む）
    

⚠️ Feedback は：

- **すべて Google に権利譲渡**
    
- 将来の製品に使われてもクレーム不可
    

---

## 5. ファインチューニングに関する扱い

これは会議内容と非常に一致しています。

- ファインチューニングは：
    
    - **Googleの計算環境上で実施**
        
    - データは Panasonic → Google にアップロード
        
- 段階制：
    
    1. ER 1.5 利用
        
    2. VLA アクセス
        
    3. ファインチューニング（関係構築後）
        

👉 **いきなりFine-tuningは不可**  
👉 **Trusted Tester → 信頼 → 次段階**

---

## 6. 秘密情報（Confidentiality）

以下は **Googleの機密情報**：

- Test Products の存在そのもの
    
- モデル仕様・性能
    
- Trusted Tester Program 参加事実
    
- 契約内容
    

🚫 社外公開・論文・発表は原則NG  
（別途書面許可が必要）

---

## 7. 安全・責任・リスク

### 7.1 Safety

- Test Products は **実験的**
    
- 誤動作・不正確な出力あり得る
    
- **安全クリティカル用途で使うな**と明記
    

### 7.2 責任の所在

- 事故・損害が起きた場合：
    
    - **Panasonic側がGoogleを補償（Indemnify）**
        
- Googleの責任上限：
    
    - **最大1万ドル**
        

👉 実証環境は  
**隔離・フェンス・人非接触が前提**

---

## 8. 契約期間・終了

- **いつでも双方解約可能**
    
- 解約後も：
    
    - データ権利
        
    - 機密保持
        
    - 責任条項  
        → **存続**
