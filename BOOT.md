# BOOT.md - 起動時コンテキスト復元

Gateway起動時に自動実行される。以下の手順で文脈を復元する。

## 復元手順

1. `memory/NOW.md` を読む — 最後の作業状態を確認
2. `MEMORY.md` を読む — 長期記憶を復元
3. 直近の日次ノート（`memory/YYYY-MM-DD.md`）を読む — 当日・前日分
4. Discordの #一般 チャンネルに復元完了を通知する

## 通知フォーマット

```
🔄 **起動しました（milo claw）**
📅 {今日の日付}
📝 直近の状況: {NOW.mdの要点1〜2行}
⚠️ 未完了: {ブロッカーや待ち事項があれば}
```

何も変化がなければ通知は最小限でよい。
