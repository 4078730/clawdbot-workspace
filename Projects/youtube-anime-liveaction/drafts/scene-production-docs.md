## Scene 001: 進撃の巨人 — エレン・イェーガー「この裏切り者がぁぁぁ!!」

### 原作シーン詳細

- **出典:** Season 2 第31話「戦士」
- **状況:** ウォール・ローゼの城壁の上。同期の戦友ライナー・ブラウンが突然「俺が鎧の巨人で、こいつが超大型巨人だ」と告白。エレンは最初「お前…疲れてるんだよ」と否認するが、ライナーが腕を上げて実力行使に出た瞬間、裏切りを確信。怒りと絶望で「この裏切り者がぁぁぁ!!」と絶叫し、自らの手を噛んで巨人化する。
- **場所:** ウォール・ローゼ城壁の上。幅10m程の石造りの壁上通路。背景に壁外の広大な平原と遠くに山脈。壁の内側には小さな街が見える。
- **天候・時間帯:** 昼間、曇天。灰色の空から薄い光が差す。強風。
- **エレンの服装:** 調査兵団制服（茶色の革ジャケット、白いシャツ、白いパンツ、ブーツ）。腰から脚にかけて立体機動装置のベルト。背中に「自由の翼」紋章入りの緑マント。

### ファーストフレーム画像の要件

- エレン正面バストアップ
- 調査兵団制服＋緑マント
- 表情: 信じられないという苦悩・目に涙が滲む
- 背景: 石造りの城壁上、曇天

### プロンプト（10秒・9:16・音声ON）

```
Shot 1: エレンの顔のクローズアップ。苦悩の表情がゆっくりと怒りに変わる。目に涙が溜まり、歯を食いしばる。風がマントと髪を激しく揺らす。曇天の城壁の上。低い風の音。
Shot 2: エレンが天を仰いで絶叫する。首の筋が浮き上がり、涙が吹き飛ぶ。城壁の上で風が渦巻く。カメラが少し引いてミディアムショット。反響する絶叫と風切り音。
Shot 3: エレンが右手を口元に持っていき、歯を食い込ませる。噛んだ瞬間、体から黄金の稲妻が爆発。蒸気が噴き出す。ローアングル見上げ。雷鳴のような轟音。
```

## Scene 002: ワンピース — モンキー・D・ルフィ「ギア5 ニカ覚醒」

### 原作シーン詳細

- **出典:** ワノ国編 アニメ第1071話 / 漫画第1044話「解放の戦士」
- **状況:** 鬼ヶ島の屋上、四皇カイドウとの死闘。ルフィはカイドウの金棒の一撃で倒され、心臓が止まる。周囲は敗北を悟る。しかし、ルフィの心臓が「ドンドットット…」と太鼓のようなリズムを刻み始める。「ゴムゴムの実」の真の名——ヒトヒトの実 幻獣種 モデル:ニカが覚醒。髪と眉が白く変色し、眉毛がくるりと渦を巻き、瞳孔が赤く光る。体から白い蒸気が立ち上り、ルフィは堪えきれないようにケタケタと笑い出す。「なんか楽しい！」——もう一度立ち上がり、自由の戦士「ニカ」として戦いを再開する。
- **場所:** 鬼ヶ島 髑髏ドームの屋上。夜空に浮かぶ満月。戦闘で崩壊した石畳の地面、瓦礫が散乱。背景に炎が燃え、黒煙が立ち上る。鬼ヶ島は空に浮いている。
- **天候・時間帯:** 深夜。満月の白い光が主光源。戦場の炎がオレンジの補助光。
- **ルフィの外見（覚醒後）:** 白い髪（逆立ち、炎のようにうねる）。服も白く変色。瞳孔が赤い。眉毛がくるりと渦巻き。体から常に白い蒸気。表情は満面の笑み——歯を見せて笑っている。サンダル、短パン、開襟シャツ（全て白）。麦わら帽子は首の後ろにぶら下がり、紐も白く変色。

### ファーストフレーム画像の要件

- ルフィが地面に倒れている（覚醒直前）→ まだ黒髪、目を閉じている
- または覚醒後の白髪ルフィが立ち上がった姿（笑顔）
- 背景: 夜空＋満月＋鬼ヶ島屋上の瓦礫＋遠くの炎

### プロンプト（10秒・9:16・音声ON）

```
Shot 1: ルフィが地面に倒れている。沈黙。やがて心臓が「ドンドットット」と太鼓のリズムを刻み始める。体から白い蒸気がゆっくり立ち上り、髪が黒から白に変色していく。見下ろしアングル。夜空の満月と瓦礫の戦場。太鼓の鼓動音だけ。
Shot 2: ルフィがむくりと起き上がる。白い髪が炎のように逆立つ。赤い瞳が見開かれ、口元がゆっくり笑みに変わり、堪えきれずにケタケタと笑い出す。正面ミディアムショット。月明かりと炎に照らされる。太鼓のリズムが加速、ルフィの笑い声。
Shot 3: ルフィが拳を天に高く掲げる。白い蒸気が爆発的に噴き出し、足元の石畳がゴムのようにうねり変形する。満月を背にしたシルエット。ローアングル見上げ。太鼓のクライマックス、地面が歪む音、解放感のある風の轟き。
```

## Scene 003: 鬼滅の刃 — 我妻善逸「雷の呼吸 漆ノ型 火雷神」

### 原作シーン詳細

- **出典:** 無限城編 / 漫画第145話「幸せの箱」
- **状況:** 無限城に落とされた善逸は、かつての兄弟子・獪岳と再会する。獪岳は上弦の陸の後任として鬼になった。善逸と獪岳は同じ師匠・桑島慈悟郎の下で雷の呼吸を学んだが、善逸は壱ノ型しか使えず、獪岳は壱ノ型以外の弐〜陸を使えた。獪岳の裏切りにより師匠は責任を取って切腹。善逸は師匠の死を背負い、**今回は眠らずに目を開いたまま**獪岳と対峙する。獪岳の雷の呼吸・弐〜陸ノ型（鬼の血鬼術で強化された黒い雷）を凌ぎ、最後に善逸が独自に編み出した型——**雷の呼吸 漆ノ型「火雷神（ほのいかづちのかみ）」**を放つ。「これは俺の型だよ」「俺が考えた俺だけの型」。霹靂一閃を極限まで研ぎ澄まし昇華させた捨て身の居合一閃。雷の龍のようなエフェクトが走り、獪岳の首を一太刀で斬り落とす。
- **場所:** 無限城の内部。エッシャーの絵のような超現実的な和風建築。無数の木造の廊下・階段・障子が重力を無視して四方八方に伸びる。薄暗い空間に無数の提灯の光がぼんやりと灯る。和風の木目の床、欄干、障子。
- **天候・時間帯:** 屋内（無限城内部）。提灯のオレンジの薄明かり。暗い。
- **善逸の外見:** 鬼殺隊の黒い隊服（詰襟）に三角柄の黄色い羽織。金髪のボブカット。**今回は目を開いている**。覚悟を決めた真剣な目。握りしめた日輪刀。体に傷と出血あり（獪岳の攻撃で負傷）。

### ファーストフレーム画像の要件

- 善逸が日輪刀を構えて立っている（目を開いている・覚悟の表情）
- 黒い隊服＋黄色い羽織、金髪
- 体に戦闘の傷・出血
- 背景: 無限城の異空間（木造の廊下が四方に伸びる、提灯の灯り）

### プロンプト（10秒・9:16・音声ON）

```
Shot 1: 善逸が目を開いたまま刀を腰に構える居合の姿勢。覚悟を決めた真剣な表情。傷だらけの体から血が滴る。黄色い羽織が微かに揺れる。無限城の異空間——木造の廊下と階段が四方に伸び、提灯がぼんやり灯る。静寂の中、善逸の呼吸音だけ。
Shot 2: 善逸の足元から黄金の雷紋が走る。「これは俺の型だ」——全身に雷が纏い、目にも留まらぬ速さで踏み込む。居合の一閃。刀を完全に振り切る。雷の龍のような巨大なエフェクトが画面を貫く。正面からの固定カメラ。雷鳴の轟音。
Shot 3: 善逸が刀を振り切った体勢で静止。体を完全に前のめりに伸ばし、刀身に黄金の雷が残光として走る。背景の無限城の構造物が斬撃の衝撃で崩れ始める。善逸の横顔クローズアップ、目に涙。雷の余韻が消え、静寂へ。
```

## プロンプト設計方針（Seedance 2.0ガイド準拠）

### カット割りの書き方
- `Shot 1:` `Shot 2:` `Shot 3:` でシーケンスを記述
- Seedance 2.0が自動でカメラ切り替え・トランジションを生成
- 各Shotは1つの被写体アクションに集中

### 構造テンプレート
```
[被写体] + [動作/状態] + [環境/背景] + [カメラワーク] + [雰囲気/スタイル] + [音声]
```

### 制約
- 語数: 各Shot 30〜50語程度（全体で100語前後）
- カメラ動作は各Shot 1つだけ
- 被写体から書き始める
- 物理描写を含める（風、蒸気、光の反射、血、衣服の揺れ）
