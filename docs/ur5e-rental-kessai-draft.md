【目的・理由】
Robotics Foundation Agent（RFA）プロジェクトにおいて、VLA（Vision-Language-Action）モデルの
実機検証および双腕マニピュレーションの学習データ取得を加速するため、

UR5eは産業標準の6軸協働ロボットであり、高精度な力制御と豊富なソフトウェアエコシステムを持ち、
VLAポリシーの実機評価・学習データ収集に最適である。
