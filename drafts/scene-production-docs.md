## Scene 001: 進撃の巨人 — エレン・イェーガー「この裏切り者がぁぁぁ!!」

### 原作シーン詳細

- **出典:** Season 2 第31話「戦士」
- **状況:** ウォール・ローゼの城壁の上。同期のライナーが突然「俺が鎧の巨人で、こいつ（ベルトルト）が超大型巨人だ」と告白。エレンは「お前…疲れてるんだよ」と信じようとするが、ライナーが実力行使に出た瞬間に裏切りを確信。怒りと絶望で「この裏切り者がぁぁぁ!!」と絶叫し、手を噛んで巨人化。
- **場所:** ウォール・ローゼの城壁上。石造りの壁面。背景に壁外の広大な平原。
- **時間帯:** 昼（曇天、薄暗い光）
- **服装:** 調査兵団の制服（茶色のジャケット、白いシャツ、立体機動装置ベルト、自由の翼マント）

### ファーストフレーム画像の要件

- エレン正面バストアップ
- 調査兵団制服＋マント着用
- 表情: 苦悩・動揺（怒りが込み上がる直前）
- 背景: 石造りの城壁の上、曇り空

### プロダクション（3クリップ構成 → 後編集で1本に）

**設定共通:** 9:16 / 音声ON / 5秒

---

#### CLIP A — 絶望から怒りへの表情変化

```
エレンの表情が苦悩から激しい怒りに変わっていく。
歯を食いしばり、目が見開かれ、顔が震える。
風がマントと髪を激しくなびかせる。
ゆっくりドリーインで顔に寄る。
曇天の城壁の上、薄暗い光。低い風の音。
```

#### CLIP B — 絶叫

```
エレンが天を仰いで絶叫する。
涙が飛び散り、首の筋が浮き上がる。
叫びの衝撃で周囲の空気が揺れるような演出。
正面バストアップの固定カメラ。
城壁の上、強風。絶叫の反響音と風切り音。
```

#### CLIP C — 手を噛む・巨人化の閃光

```
エレンが自分の右手に歯を食い込ませる瞬間。
噛んだ直後、体から黄金の稲妻が爆発的に走る。
蒸気が噴き出し、シルエットが巨大化し始める。
ローアングルの見上げショット。
曇り空に黄金の閃光。雷鳴のような轟音と蒸気の噴出音。
```

## Scene 002: ワンピース — モンキー・D・ルフィ「ギア5 ニカ覚醒」

### 原作シーン詳細

- **出典:** ワノ国編 アニメ第1071話 / 漫画第1044話「解放の戦士」
- **状況:** 鬼ヶ島の屋上でカイドウに敗北し、心臓が止まる。しかし「ゴムゴムの実」の真の姿——ヒトヒトの実 幻獣種 モデル:ニカが覚醒。髪と服が白く変わり、眉毛がくるりと巻き、瞳が赤く光る。心臓が「ドンドットット」と太鼓のリズムを刻む。ルフィはケタケタと笑い始め、自由な力を解放する。
- **場所:** 鬼ヶ島 屋上（夜空、満月、瓦礫散乱、炎が燃える戦場）
- **時間帯:** 深夜（月明かり＋炎のオレンジ光）
- **外見（覚醒後）:** 白い髪（炎のように逆立つ）、白い服、赤い瞳、巻き眉、体から白い蒸気、笑顔

### ファーストフレーム画像の要件

- ルフィ（覚醒直前：倒れている状態 or 覚醒後：白髪で笑顔）
- 背景: 夜空＋満月＋鬼ヶ島屋上の瓦礫

### プロダクション（3クリップ構成 → 後編集で1本に）

**設定共通:** 9:16 / 音声ON / 5秒

---

#### CLIP A — 心臓の太鼓・覚醒の始まり

```
倒れたルフィの体から白い蒸気がゆっくり立ち上る。
髪が黒から白に変色していく。
心臓のリズムに合わせて体が微かに脈動する。
見下ろしアングルの固定ショット。
夜空と満月の背景。「ドンドットット」と太鼓の音だけ。
```

#### CLIP B — 起き上がり・笑い出す

```
ルフィがむくりと起き上がる。
白い髪が炎のように逆立ち、赤い瞳が見開かれる。
口元がゆっくり笑みに変わり、ケタケタと笑い出す。
正面ミディアムショット、固定カメラ。
月明かりと炎に照らされた戦場。太鼓の加速と笑い声。
```

#### CLIP C — ニカの力解放

```
ルフィが拳を天に向かって高く掲げる。
白い蒸気が爆発的に噴き出し、地面がうねるように変形する。
満月を背にした力強いシルエット。
ローアングル見上げショット。
太鼓のクライマックス、地面が割れる轟音、解放の風。
```

## Scene 003: 鬼滅の刃 — 我妻善逸「雷の呼吸 壱ノ型 霹靂一閃・六連」

### 原作シーン詳細

- **出典:** 立志編 第17話「ひとつのことを極め抜け」（那田蜘蛛山編）
- **状況:** 那田蜘蛛山で蜘蛛の鬼（兄）に毒を注入され、体が蜘蛛に変わり始めている。恐怖で泣き叫ぶが、あまりの恐怖に気絶し眠りに落ちる。眠った瞬間、人格が一変。怯えは消え、研ぎ澄まされた剣士の目になる。師匠・桑島慈悟郎の言葉「ひとつのことしかできないならそれを極め抜け。極限の極限まで磨け」を思い出し、唯一の技「雷の呼吸 壱ノ型 霹靂一閃」を六連続で放ち、鬼の首を斬り落とす。
- **場所:** 那田蜘蛛山の森の中。蜘蛛の巣が張り巡らされた暗い森。木々の隙間から月明かり。
- **時間帯:** 夜（月明かりがスポットライトのように一筋差す）
- **外見:** 鬼殺隊の黒い隊服に黄色い羽織。金髪ボブカット。目を閉じている（睡眠状態）。腰に刀を帯び、居合の構え。

### ファーストフレーム画像の要件

- 善逸が居合の構えで立つ（目を閉じている）
- 黒い隊服＋黄色い羽織
- 背景: 暗い森、蜘蛛の巣、上から月明かりが差し込む
- 表情: 眠り（静かだが研ぎ澄まされた空気感）

### プロダクション（3クリップ構成 → 後編集で1本に）

**設定共通:** 9:16 / 音声ON / 5秒

---

#### CLIP A — 眠りの静寂・居合の構え

```
善逸が目を閉じたまま居合の構えで佇んでいる。
風が黄色い羽織と金髪をわずかに揺らす。
月明かりが上から一筋差し込み、善逸だけを照らす。
ゆっくりドリーインで全身から上半身に寄る。
蜘蛛の巣だらけの暗い森。静寂、呼吸音、微かな雷鳴。
```

#### CLIP B — 雷紋・抜刀の瞬間

```
善逸の足元から黄金の雷紋が地面を走る。
一瞬の溜めの後、目にも留まらぬ速さで抜刀する。
雷光が刀身を包み、黄金の軌跡が画面を横切る。
正面固定カメラ、抜刀のスローモーション。
暗い森に黄金の閃光。雷鳴のような鋭い抜刀音。
```

#### CLIP C — 霹靂一閃・六連の雷跡

```
善逸が雷速で六方向に斬撃を繰り出す。
黄金の雷の軌跡が六本の線となって暗闇を切り裂く。
最後に元の位置で刀を鞘に納める。カチンと納刀音。
ワイドショット、見上げアングル。
蜘蛛の巣が一斉に切断され崩れ落ちる。連続雷鳴から静寂へ。
```

## プロンプト設計方針（Seedance 2.0ガイド準拠）

各プロンプトは以下の構造テンプレートに従う:

```
[被写体] + [動作/状態] + [環境/背景] + [カメラワーク] + [雰囲気/スタイル] + [音声]
```

### 制約
- **1プロンプト = 1連続クリップ**（カット割りは後編集）
- **語数: 30〜100語**
- **カメラ動作は1つだけ**
- **被写体から書き始める**
- **物理描写を含める**（風、煙、光の反射、衣服の揺れ）
